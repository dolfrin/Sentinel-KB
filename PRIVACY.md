# Privacy Policy

**Sentinel-KB** — Last updated: April 13, 2026

## What Sentinel-KB does

Sentinel-KB is a security vulnerability scanner that runs as a Claude Code plugin. It performs static code analysis locally and queries a hosted knowledge base API for vulnerability context.

## Data collection

### Static scan (local)
- Runs entirely on your machine
- No code, files, or scan results leave your device
- No data is collected, stored, or transmitted

### Knowledge base API (kb.quantumwing.io)
- When you use KB search or stats, the plugin sends **search queries** (keywords, category filters) to our API
- We log: request IP, timestamp, endpoint path, and response time — for rate limiting and abuse prevention only
- We do **not** log search query content
- We do **not** collect, store, or transmit any of your source code
- No cookies, no tracking, no analytics

## Data storage

- Logs are retained for 30 days, then deleted
- No personal data is stored beyond server access logs
- No user accounts or registration required

## Third parties

- We do not sell, share, or transfer any data to third parties
- The API runs on infrastructure operated by the project maintainer
